JULIA Commands
===================================
ctrl+l - To clear the Screen
ctrl+d - To exit the Screen
ctrl+c - To stop the Long running process
workspace() - To start a New Session
? - for Help about Julia - type sort
]? - for pkg Help about Julia - type sort
